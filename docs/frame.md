# The `<frame>` Tag

## **Purpose**

Defines a visual frame to be rendered.

## **Attributes**

| Attribute             | Description                      |
| --------------------- | -------------------------------- |
| `duration` (optional) | Duration of the frame in seconds |

## **Example**

```html
<frame duration="3s"><div>Hello</div></frame>
```
