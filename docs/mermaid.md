# The `<mermaid>` Tag

## **Purpose**

Embeds Mermaid.js diagrams in YTML scripts.

## **Example**

```html
<composite id="diagram">
  <mermaid>
    graph TD A[User] --> B[Frontend] B --> C[Backend] C --> D[Database]
  </mermaid>
</composite>
```
